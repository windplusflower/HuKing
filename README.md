# Hu King - 胡王 Mod

[**English Version**](README-en.md) | **中文说明**

![胡长老](HuKing.png)

## 简介
**Hu King** 是一个针对《空洞骑士》中“胡长老”战役的完全重制 Mod。该 Mod 将原本单调的 Boss 战重构为一场融合了多种经典游戏机制、极具创意且充满挑战的多阶段战斗。

## 核心技能介绍
在这个 Mod 中，胡长老拥有五种截然不同的攻击模式：

1. **电锯迷宫 (Sawblade Maze)**
   - 使用大量电锯在场中组成复杂的迷宫，考验玩家的位移精度和反应速度。
2. **激光飞行 (Bird Style & Lasers)**
   - 玩家需要上下穿梭，钻过上下摆动柱子的缝隙，同时躲避致命的**黄色粗激光**打击。
3. **飞针方阵 (Needle Square)**
   - 生成一个方框封锁小骑士的行动区域，并从四面八方发射密集的**飞针**攻击。
4. **三维弹球 (Pinball)**
   - 发射在墙壁间高速反弹的电锯，击中Boss并防止自己被击中。
5. **三消挑战 (Match-Three)**
   - **机制**：下落带有亮红、亮黄、亮蓝三种颜色的电锯。
   - **操作**：玩家需打击下落电锯，将其平滑吸附到右侧的格点墙中。
   - **消除**：当三个相同颜色的电锯相邻时，会触发消除，打通前往boss的道路。
6. 本mod对小骑士作了一些限制，不能使用剑技，法术无法造成伤害。
7. 调谐血量为10，进升、辐辉为15。

## 安装要求
⚠️ **重要事项：**
- 本 Mod 依赖 **[Satchel](https://github.com/PrashantMohta/Satchel)** 插件库。
- **必须使用较新版本的 Satchel**。由于 Mod 使用了较新的 API，旧版本 Satchel 会导致技能无法正常触发或游戏崩溃。
